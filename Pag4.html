¿Que es wampserver?

¡Aprendamoslo!

Traducción del inglés-WampServer se refiere a una pila de soluciones para el sistema operativo Microsoft Windows, creada por Romain Bourdon y que consta del servidor web Apache, OpenSSL para soporte SSL, base de datos MySQL y lenguaje de programación PHP.
