¿Qué es el cableado horizontal?

Descripcion:

El cableado horizontal es una parte fundamental de la infraestructura de redes, que se refiere a la instalación de cables de red que se extienden desde el panel de conexión o el armario de telecomunicaciones hasta las estaciones de trabajo o dispositivos finales en un entorno de oficina o edificio.

Características del cableado horizontal:

Distancia:

Generalmente, se extiende hasta 90 metros desde el panel de parcheo hasta el punto de conexión final.

Tipos de cables:

Usualmente se utilizan cables de par trenzado (como UTP) o cables de fibra óptica.

Seccion 7(

¿Qué es el cableado vertical o backbone?

Descripcion:

El cableado vertical, también conocido como backbone, es la parte de la infraestructura de red que conecta diferentes pisos o áreas de un edificio, permitiendo la interconexión de redes locales. Este tipo de cableado es fundamental para la comunicación entre diferentes segmentos de la red.

Seccion 8( Tipos de cable para el sistema de cableado horizontal

Cable UTP (Unshielded Twisted Pair)

Es el tipo más común de cable para redes Ethernet. Está compuesto por pares de hilos trenzados que ayudan a reducir la interferencia electromagnética. Hay diferentes categorías, como Cat5e, Cat6 y Cat6a, que ofrecen diferentes capacidades de velocidad y distancia.

Cable STP (Shielded Twisted Pair)

Similar al UTP, pero con un blindaje adicional que protege contra interferencias externas. Es más costoso que el UTP y se utiliza en entornos con alta interferencia electromagnética.

Cable de Fibra Óptica

Utiliza filamentos de vidrio o plástico para transmitir datos mediante luz. Ofrece alta velocidad y ancho de banda, así como una distancia de transmisión sin pérdida de señal.

Cable Coaxial

Consiste en un conductor central rodeado por un aislamiento, una malla de blindaje y una cubierta exterior. Aunque es menos común en redes LAN modernas, todavía se utiliza en algunas aplicaciones específicas.

Para que se utiliza un cable directo en la red

Descripción:

En general, un cable cruzado se utiliza para conectar dos dispositivos del mismo tipo, como por ejemplo un PC a una PC o un switch a otro switch. Por otro lado, el cable directo conecta dos dispositivos diferentes entre sí, como por ejemplo un PC y un switch.
